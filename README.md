# TicTacToe
A simple yet engaging implementation of the classic Tic Tac Toe game using HTML, CSS, JavaScript. This project allows two players to compete against each other in a turn-based format, aiming to align three symbols in a row, column, or diagonal.

# Features
- Theme Toddle: Player can Switch Between two themes i.e ?Dark:LIGHT
- Reset Option: To Reset the Game and Play a Fresh.

# Technologies
- HTML, CSS, JavaScript
